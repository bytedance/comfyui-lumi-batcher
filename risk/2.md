描述：
【漏洞标题】comfyui-lumi-batcher 存在目录穿越漏洞

【漏洞描述】攻击者可以在无需认证的情况下，通过构造恶意 HTTP 请求读取 ComfyUI 服务器上的敏感文件（包括数据库、用户上传的资源文件等）
comfyui-lumi-batcher版本：1.0.11
comfyui版本：0.24.1

影响文件：lumi_batcher_service/common/file_path.py:54-115、lumi_batcher_service/handler/batch_tools.py:456-523

问题描述：
is_under_delete_white_dir() 函数用于校验文件路径是否在允许的目录范围内。其校验逻辑是：提取白名单目录的 "最后一级文件夹名称"，然后检查该名称是否出现在用户传入路径的任意位置。虽然函数使用了 Path.resolve() 来消除 ../ 穿越，但在白名单目录之间进行路径穿越时（如 output → input、download → db），resolve 后的路径仍然包含白名单关键字，从而绕过检查。

白名单当前包含以下目录名：comfyui-lumi-batcher、comfyui_lumi_batcher_workspace、input、output。攻击者不能逃逸到 ComfyUI 目录之外，但可以在白名单目录之间自由穿越。

受影响接口：

GET /api/comfyui-lumi-batcher/view-image?type=output&file_name=../input/<任意文件>

构造路径：{output_dir}/../input/<file> → resolve 后 → {comfyui_dir}/input/<file>
input 在白名单中 → 校验通过 → 返回文件内容
GET /api/comfyui-lumi-batcher/view-image?type=download&file_name=../db/batch_tools.db

构造路径：{workspace}/download/../db/batch_tools.db → resolve 后 → {workspace}/db/batch_tools.db
comfyui_lumi_batcher_workspace 在白名单中 → 校验通过 → 返回数据库文件
